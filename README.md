# transferImageBySocket
# 简介
这是一个基于Gradle的socket项目。


# 环境
<ul>
<li>JDK 1.8</li>
<li>IntelliJ Idea 2020</li>
</ul>

# 功能
服务端读取图片发送给客户端，客户端保存图片到本地

# 目录结构
```bash
├── Readme.md                   // help
├── src        
│   └── main
│       └── java     
│           ├── Client      	// 客户端
│           └── Server          // 服务端
│   ├── 1.png                   // 服务端读取
│   └── client.png      	    // 客户端保存
```
